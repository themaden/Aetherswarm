# AetherSwarm Frontend (Command Center) 🚀

A high-fidelity "Black-Box" dashboard for managing the AetherSwarm ecosystem. Designed with a premium, terminal-grade aesthetic to impress judges and users alike.

## 🛠 Tech Stack

- **Framework:** Next.js 14+ (App Router)
- **Styling:** Tailwind CSS with custom glassmorphism and neon-glow utility system.
- **Web3:** Wagmi, Viem, and ConnectKit for seamless wallet integration.
- **Icons:** Lucide React.

## 🚀 Development

1. **Install Dependencies:**
   ```bash
   npm install
   ```

2. **Start Development Server:**
   ```bash
   npm run dev
   ```

3. **Open Dashboard:**
   Navigate to [http://localhost:3000](http://localhost:3000).

## ✨ Features

- **Real-time Terminal:** Streams live logs from the backend AI Enclave.
- **Mesh Map:** Visualizes discovered libp2p nodes in the Ghost Swarm.
- **iNFT Gallery:** Shows AI model ownership via ERC-7857 standard.
- **Vault Interface:** High-end UI for asset management and autonomous hook performance tracking.
